📝  ToDo Uygulaması

Bu uygulama, kullanıcıların günlük görevlerini oluşturabileceği, açıklama ekleyebileceği ve görevleri tamamlayabileceği basit ama modern bir yapılacaklar listesi uygulamasıdır.



🚀 Özellikler:

- Görev oluşturma
- Görev düzenleme
- Açıklama ekleme (Modal ile)
- Görev silme

 🛠 Teknolojiler:
 - React Native + Expo
- TypeScript
- UUID v4
